# Ofeianht Skincare With ReactJS Redux

# Features: 
         + Add to cart, remove from cart, adjust quantity.
         + Compare products,product search, detail item.
         + Shop, loadmore items, search by price.
         + 5 Pages: home, shop, contact, about, blog.
         + Validate form,...
